G.I.R.A.F.E. Graphical Interface for Representing Automatic Function Embedding

This tool is targeted as Internet Service Providers that are willing to deploy a third-party VCDN Services targetting OTT-VOD delivery market. The algorithm we implemented deploys a service function chain composed of 2 VNF: the Virtual Media Gateways (routing) and the Virtual CDN cache (content delivery) that garanty the Quality of Service for end-users on enhanced paths.

After loading a demo topology, we can configure the User pools which consumes the data, as well as the legacy CDN network peering points, which will support the best-effort IP traffic. Other Service Level agreement parameters can be configured. Finally, we can compute the optimal embedding or manually set the vNF numbers.
