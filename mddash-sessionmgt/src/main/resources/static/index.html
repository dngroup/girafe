Testing the demo

  1. Open the developer tools windows in chrome (up right box) or use the following shortcut:"Ctrl + Shift + i"
  2. Go to the tab "Network"
  3. Choose your demo in the "stream" tab
  4. Click on "Load"
  5. You should be able to see the different video segments being downloaded in the network panel. You can also check whether the segment are properly being downloaded or if the requests are being dropped due to timeout expiration.
  6. if you want to try the same demonstration again, click on"disable cache" in "developer tools/network" panel.

Expected result

According to the number of description segments being retrieved for one video period, the displayed quality is to change over time.

If all three descriptions are collected, the best video quality is to be displayed.

Should some description segments be missing, the displayed quality should constantly change (on a GOP basis) for the duration of the video segment.
